## Aturan Glosarium

1. **Konsultasi WAJIB** sebelum menerjemahkan istilah berulang
2. **Istilah baru** harus ditambah ke glosarium yang relevan + commit dengan `glossary` type
3. **Perubahan** entry yang ada harus didiskusikan via issue
4. **Format:** `Inggris | Indonesia | Catatan/Konteks`
5. **Konsistensi adalah hukum** — jangan terjemahkan istilah berbeda di file berbeda

## Format Penambahan Entry Baru

Saat menambah entry baru:

```markdown
| Original | Translation | Catatan |
|----------|-------------|---------|
| Word | Terjemahan | Konteks/alasan/contoh |
```

**Selalu sertakan catatan** jika:
- Terjemahan tidak literal
- Ada pengecualian konteks
- Ada alternatif yang dipertimbangkan tapi ditolak
- Referensi historis/religius

Setelah update, commit dengan format:
```
glossary(category): tambah X istilah baru

- istilah1: Inggris → Indonesia (catatan)
- istilah2: ...

Refs: file source yang berisi istilah ini
```
