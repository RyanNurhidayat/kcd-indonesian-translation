## Aturan Saat Menemukan Karakter Baru

Saat menerjemahkan dialog, kalau ketemu karakter yang **belum di glossary ini**:

1. **Identifikasi** dari dialog ID (speaker code)
2. **Cek** di KCD Wiki (https://kingdom-come-deliverance.fandom.com/) untuk role mereka
3. **Tambahkan entry** di sini dengan format:
   ```
   | Original | Indonesia | Peran/Catatan |
   ```
4. **Update** di character bible jika ada voice/style note
5. **Commit terpisah** dengan tipe `glossary`:
   ```
   glossary(characters): tambah X karakter dari quest Y
   ```
