## Title + Name Patterns

### "Sir [Name]" Pattern
- **Sir Radzig** → tetap "Sir Radzig" (Sir dipertahankan)
- **Sir Hans Capon** → tetap "Sir Hans Capon"
- **Sir Henry of Leipa** → "Sir Henry dari Leipa"

### "Lord [Name]" Pattern (Feudal)
- **Lord Divish** → "Tuanku Divish" atau "Lord Divish"
- **Lord Hanush** → "Tuanku Hanush"
- **Lord Capon** → "Tuanku Capon"
- **Lord Mayor** → "Tuan Walikota"

### "Lord [Name]" Pattern (Religious)
- **Lord God** → "Tuhan Allah"
- **Lord Almighty** → "Tuhan Mahakuasa"
- **Lord Jesus** → "Tuhan Yesus"

### "Lady [Name]" Pattern
- **Lady Stephanie** → "Tuan Putri Stephanie" (bangsawan)
- **Lady Ophelia** → "Tuan Putri Ophelia"

### "Master [Name]" Pattern (Craftsmen)
- **Master Blacksmith** → "Tuan Pandai Besi"
- **Master Builder** → "Tuan Tukang Bangun"
- **Master Huntsman** → **"Kepala Pemburu"** atau "Master Huntsman" (lebih historis)

### "Father [Name]" Pattern (Priests)
- **Father Simon** → "Pater Simon"
- **Father Godwin** → "Pater Godwin"

### "Brother [Name]" Pattern (Monks)
- **Brother Nicodemus** → "Bruder Nicodemus"
- **Brother Cyril** → "Bruder Cyril"

## "Lord" — Critical Disambiguation

Ini istilah TERPENTING untuk dibedakan. **"Lord"** punya 3 arti di KCD:

### 1. Lord = Tuhan (Religious)

```
"The Lord is my shepherd."
→ "Tuhan adalah gembalaku."

"Lord, have mercy."
→ "Tuhan, kasihanilah."

"Praise the Lord!"
→ "Pujilah Tuhan!"
```

### 2. Lord = Tuanku (Feudal)

```
"Lord Capon arrived."
→ "Tuanku Capon telah tiba."

"My Lord, may I speak?"
→ "Tuanku, bolehkah hamba berbicara?"

"He's the Lord of Rattay."
→ "Beliau adalah Tuanku Rattay."
```

### 3. Lord = Tuan/Tuanku (Civic)

```
"Lord Mayor, please."
→ "Tuan Walikota, silakan."
```

### Cara Menentukan

Lihat konteks:
- Ada "**God**" atau "**heaven**" atau dalam doa → Religious
- Ada nama setelahnya (e.g., "Lord [Name]") → biasanya Feudal
- "**Lord Mayor**" → Civic

## Aturan Saat Menemukan Gelar/Sapaan Baru

1. **Identifikasi kategori**: Feodal / Religius / Militer / Sipil / Generic
2. **Cek context**: Siapa yang dipanggil? Siapa yang menyapa?
3. **Tambahkan entry** di kategori yang tepat
4. **Default untuk gelar feodal**: "Tuanku" untuk Lord, "Sir" untuk Sir
5. **Commit** dengan tipe `glossary(titles-honorifics)`
